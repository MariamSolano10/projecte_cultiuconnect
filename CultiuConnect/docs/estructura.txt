Directoris principals
---------------------

- `api/`
  - Endpoints JSON per a funcionalitats AJAX i suport GIS.
- `assets/`
  - `img/` per logos i fons.
  - `fotos/parceles/` i `fotos/varietats/` per imatges de suport.
- `auth/`
  - Login i logout.
- `config/`
  - `config.example.php`
  - `db.php`
  - `db_config.php` (configuracio local, fora de control de versions)
- `css/`
  - `estils_globals.css`
  - `estils_v2.css`
- `data/geometries/`
  - Backups JSON de parceles, sectors i infraestructures.
- `database/`
  - `01_cultiuconnect_bd.sql`
  - `02_cultiuconnect_inserts.sql`
  - `03_lines_corrected.sql`
- `docs/`
  - PDFs d'especificacio del projecte.
  - Documents d'auditoria i suport.
  - Aquest fitxer `estructura.txt`.
- `errors/`
  - Pagines 403, 404 i 500.
- `includes/`
  - `header.php`
  - `footer.php`
  - `helpers.php`
- `js/`
  - `scripts.js`
- `libs/`
  - `fpdf/`
  - `phpqrcode/`
- `modules/`
  - Moduls funcionals de l'aplicacio.
- `tests/`
  - `README.md` amb l'estat actual de la validacio.
- `tools/`
  - `restaurar_geometries.php`
